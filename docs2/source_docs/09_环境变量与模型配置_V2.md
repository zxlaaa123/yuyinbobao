# 09_环境变量与模型配置_V2

# AI 知识点学习与音频播报系统 - V2 环境变量与模型配置

## 1. 文档目的

本文档定义 V2 新增配置项。

V2 不替换 V1 配置。

V2 在 V1 基础上新增：

```text
AI 长文本分段配置
AI JSON 修复配置
复习计划配置
TTS 批量音频配置
导出配置
```

---

## 2. 配置原则

继续沿用 V1：

```text
敏感配置放 .env
用户可调配置放 app_settings
SettingsView 读取和保存 app_settings
后端统一通过 setting_service 读取配置
前端不直接读取敏感环境变量
```

V2 新增原则：

```text
1. 所有批量操作必须有上限配置。
2. 所有布尔配置统一保存为 "true" / "false"。
3. 所有数值配置后端必须校验范围。
4. app_settings 优先级高于 .env。
5. API Key 返回前端时必须脱敏。
```

---

## 3. 配置来源

优先级：

```text
app_settings 表
↓
.env
↓
代码默认值
```

说明：

```text
用户在设置页保存后，优先使用 app_settings。
没有保存过时，读取 .env。
.env 也没有时，使用安全默认值。
```


# 6. AI V2 配置

## 6.1 AI_ENABLE_LONG_TEXT_SPLIT

说明：

```text
是否启用长文本自动分段。
```

默认：

```text
true
```

可选值：

```text
true
false
```

使用位置：

```text
/api/materials/import-and-extract
text_splitter.py
```

---

## 6.2 AI_MAX_SEGMENTS

说明：

```text
单次资料最多分段数。
```

默认：

```text
10
```

建议范围：

```text
1 到 20
```

超过范围：

```text
后端使用默认值或返回设置错误。
```

---

## 6.3 AI_SEGMENT_SIZE

说明：

```text
每个 AI 分段最大字符数。
```

默认：

```text
7000
```

建议范围：

```text
1000 到 12000
```

说明：

```text
该值不是模型 token 数，只是本地字符切分阈值。
```

---

## 6.4 AI_ENABLE_JSON_REPAIR

说明：

```text
AI JSON 解析失败时，是否自动修复一次。
```

默认：

```text
true
```

规则：

```text
最多修复 1 次。
不能无限重试。
```

---

## 6.5 AI_SAVE_RAW_RESPONSE

说明：

```text
是否保存 AI 原始返回用于排错。
```

默认：

```text
false
```

安全要求：

```text
不能保存 API Key。
不能保存 Authorization 请求头。
默认关闭。
```

---

## 6.6 QUESTION_GENERATE_MAX_COUNT

说明：

```text
单次最多生成题目数量。
```

默认：

```text
20
```

建议范围：

```text
1 到 50
```

---

## 6.7 FLASHCARD_GENERATE_MAX_COUNT

说明：

```text
单次最多生成闪卡数量。
```

默认：

```text
10
```

建议范围：

```text
1 到 30
```

---

# 7. 复习计划配置

## 7.1 REVIEW_DAILY_LIMIT

说明：

```text
每日生成复习任务上限。
```

默认：

```text
30
```

建议范围：

```text
1 到 200
```

前端：

```text
生成今日复习时 limit 不能超过该值。
```

后端：

```text
即使前端传大值，也必须强制限制。
```

---

## 7.2 REVIEW_INCLUDE_MASTERED_WRONG

说明：

```text
生成复习任务时是否包含已掌握错题。
```

默认：

```text
false
```

建议：

```text
默认不包含，避免复习任务过多。
```

---

## 7.3 REVIEW_PRIORITY_HIGH_IMPORTANCE

说明：

```text
是否优先把 importance=high 的知识点加入复习。
```

默认：

```text
true
```

---

## 7.4 REVIEW_OVERDUE_FIRST

说明：

```text
生成和展示复习任务时是否优先逾期任务。
```

默认：

```text
true
```

---

# 8. TTS 批量配置

## 8.1 TTS_BATCH_MAX_POINTS

说明：

```text
批量音频单次最多知识点数量。
```

默认：

```text
20
```

建议范围：

```text
1 到 50
```

使用位置：

```text
/api/tts/generate-batch
/api/tts/generate-wrong-questions
```

---

## 8.2 TTS_BATCH_MAX_CHARS

说明：

```text
批量音频播报文本最大字符数。
```

默认：

```text
12000
```

建议范围：

```text
1000 到 30000
```

超过时返回：

```json
{
  "detail": "批量音频文本过长，请减少知识点数量"
}
```

---

## 8.3 TTS_SPEED

说明：

```text
TTS 语速。
```

默认：

```text
1.0
```

建议范围：

```text
0.5 到 2.0
```

注意：

```text
不同 TTS Provider 对语速参数支持不同。
不支持时可以忽略，但设置仍保留。
```

---

## 8.4 TTS_FORMAT

说明：

```text
音频格式。
```

默认：

```text
mp3
```

可选：

```text
mp3
wav
```

V2 初期推荐只使用：

```text
mp3
```

说明：

```text

```

默认：

```text
5000
```

建议范围：

```text

```

```text

或
返回错误要求缩小筛选范围
```

```text

```

## 9.2 EXPORT_INCLUDE_EXPLANATION

说明：

```text

```

默认：

```text
true
```

# 10. 设置页字段

SettingsView 新增分区：

```text

```

## 10.1 AI 增强设置

```text
启用长文本分段 AI_ENABLE_LONG_TEXT_SPLIT
```

```text
每日复习上限 REVIEW_DAILY_LIMIT
包含已掌握错题 REVIEW_INCLUDE_MASTERED_WRONG

逾期任务优先 REVIEW_OVERDUE_FIRST
```

## 10.3 TTS 批量设置

```text
批量最大知识点数 TTS_BATCH_MAX_POINTS
批量最大文本长度 TTS_BATCH_MAX_CHARS
语速 TTS_SPEED
格式 TTS_FORMAT
```

## 10.4 导出设置

```text
最大导出行数 EXPORT_MAX_ROWS

```

# 11. 后端配置读取

继续通过：

```text
setting_service.py
```

建议新增工具函数：

```python
def get_bool_setting(db, key: str, default: bool) -> bool:
    ...

def get_int_setting(db, key: str, default: int, min_value: int, max_value: int) -> int:
    ...

def get_float_setting(db, key: str, default: float, min_value: float, max_value: float) -> float:
    ...

def get_str_setting(db, key: str, default: str) -> str:
    ...
```

目的：

```text
避免每个 service 自己解析字符串。
```

# 15. Windows 使用流程

V2 不改变启动方式。

配置步骤：

```text
1. 复制 .env.example 为 .env
2. 填写 AI / TTS 基础配置
3. 启动后端
4. 启动前端
5. 打开设置页
6. 调整 V2 配置
7. 保存设置
8. 测试 AI / TTS
```

---

# 16. 配置模块验收标准

通过标准：

```text
1. .env.example 包含 V2 新配置
2. SettingsView 能展示 V2 新配置
3. 保存设置后写入 app_settings
4. app_settings 优先级高于 .env
5. 布尔值能正确解析
6. 数值字段有范围校验
7. API Key 仍然脱敏
8. 批量音频读取 TTS_BATCH_MAX_POINTS
9. 复习任务生成读取 REVIEW_DAILY_LIMIT
10. 长文本分段读取 AI_MAX_SEGMENTS 和 AI_SEGMENT_SIZE
```

不通过条件：

```text
1. API Key 明文返回前端
2. 无效数值导致后端崩溃
3. 批量接口没有读取上限配置
4. app_settings 保存后不生效
5. .env 被提交到 git
```
