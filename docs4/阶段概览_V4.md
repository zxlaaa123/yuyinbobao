# V4 阶段概览

## 阶段定位

V4 是学习效果增强阶段。目标是围绕 AI 日志、间隔重复复习、多题型生成、练习会话、PDF 导入和导出能力继续扩展。

## 已规划内容

V4 覆盖：

- AI 日志增强。
- 间隔重复复习。
- 答题结果联动复习计划。
- 首页复习概览。
- 多题型生成和刷题 UI。
- 练习会话与练习结束页。
- PDF 导入。
- 导出能力。
- 音频播放列表和批量生成增强。
- 日志脱敏和安全规则。

## 后续 AI 需要知道

- V4 必须继续回归 V1/V2/V3，不破坏已有主流程。
- AI 日志要有脱敏规则，不能保存真实 API Key。
- 间隔重复先做可解释、可验收的简单规则。
- 多题型生成要保证 JSON 可解析，失败不能写入脏数据。
- PDF、导出、播放列表属于增强能力，不能影响基础学习闭环。
