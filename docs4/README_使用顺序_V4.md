# AI Study Cast - V4 文档使用顺序

## 1. V4 定位

V4 目标是把 V3 的可用系统升级为长期学习系统。

主线：

```text
资料导入 → AI 提取 → 多题型练习 → 错题沉淀 → 间隔复习 → 音频复习 → AI 日志排障 → 数据导出
```

核心优先级：

```text
AI 日志增强 > 间隔重复 > 多题型 > 会话总结 > PDF 导入 > CSV 导出 > 音频播放增强
```

## 3. 开发规则

必须遵守：

```text
1. 每次只做一个阶段。
2. 不要一次性实现整个 V4。
3. 不要重写 V1/V2/V3 已有可运行代码。
4. 不要引入登录、多用户、云同步。
5. 不要引入 Docker、Redis、Celery、消息队列。
6. 不要把 API Key 写进代码、日志、前端。
7. 所有 AI / TTS 调用仍必须走后端。
8. 先保证能运行，再增强体验。
```
