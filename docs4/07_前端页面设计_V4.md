# AI Study Cast - V4 前端页面设计

## 1. 前端原则

继续使用：

```text
Vue 3
Vite
TypeScript
Element Plus
Vue Router
Pinia
Axios
```

页面必须优先可用。

## 5. AI 日志页

功能：

```text
日志列表
按状态筛选
按调用类型筛选
查看详情
复制错误信息
删除日志
```

字段：

```text
调用类型
模型
状态
耗时
JSON 解析状态
错误类型
创建时间
```

---

## 6. 刷题页多题型 UI

题型 UI：

```text
single_choice：单选按钮
multiple_choice：多选框
true_false：判断按钮
fill_blank：输入框
short_answer：文本框 + 查看参考答案
```

简答题：

```text
用户自己对照参考答案。
不调用 AI 自动判分。
```

## 9. Loading 与错误提示

必须有 loading：

```text
查询 AI 日志
删除 AI 日志
生成多题型题目
提交练习
生成会话总结
导入 PDF
导出 CSV
批量生成音频
标记已听
```

必须有中文错误提示：

```text
AI 日志加载失败
题型不支持
没有待复习知识点
PDF 文本提取失败
CSV 导出失败
批量音频生成失败
播放列表为空
```
