# 10_验收清单_V4

# AI Study Cast - V4 验收清单

## 1. 验收原则

V4 验收必须确认：

```text
1. V4 新功能可用。
2. V1/V2/V3 核心流程未破坏。
3. 前端能启动。
4. 后端能启动。
5. 数据库旧数据兼容。
6. 错误提示为中文。
```

---

## 2. 启动检查

后端：

```bash
cd backend
.venv\Scripts\activate
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

前端：

```bash
cd frontend
npm run dev
```

检查：

```text
GET /api/health 返回 status ok
前端 http://localhost:5173 可访问
```

---

## 3. V1/V2/V3 回归检查

必须通过：

```text
新建知识库
导入资料
AI 提取知识点
查看知识点
AI 生成题目
刷题
错题本
生成音频
播放音频
备份恢复
本地搜索
```

---

## 4. AI 日志验收

必须通过：

```text
AI 成功调用能写入日志
AI 失败调用能写入日志
JSON 解析失败能写入日志
日志列表能筛选
日志详情能查看
错误信息能复制
日志能删除
API Key 不出现在日志中
API Key 不出现在前端响应中
```

---

## 5. 复习计划验收

必须通过：

```text
新知识点有默认复习状态
今日待复习能展示
逾期复习能展示
薄弱知识点能展示
答对后 next_review_at 延后
答错后 next_review_at 提前
答错题进入错题本
首页复习概览正确
```

---

## 6. 多题型验收

必须通过：

```text
single_choice 能生成和作答
multiple_choice 能生成和作答
true_false 能生成和作答
fill_blank 能生成和作答
short_answer 能展示参考答案
旧题目仍能展示和作答
AI 输出不合法时不保存脏数据
```

---

## 7. 学习会话验收

必须通过：

```text
提交练习后生成会话记录
会话记录包含总题数、正确数、错误数、正确率
会话详情能查看题目和答案
练习结束页能展示薄弱知识点
错题重练入口可用
薄弱知识点重练入口可用
```

---

## 8. PDF 导入验收

必须通过：

```text
文本型 PDF 能导入
导入后能生成资料记录
导入后能继续 AI 提取知识点
扫描 PDF 给出中文错误
未选择知识库给出中文错误
```

---

## 9. CSV 导出验收

必须通过：

```text
全部题目能导出 CSV
按知识库导出 CSV
错题能导出 CSV
薄弱知识点能导出 CSV
CSV 能用表格软件打开
CSV 不包含 API Key
```

---

## 10. 音频增强验收

必须通过：

```text
音频播放列表能加载
能按知识库筛选
能按已听状态筛选
能倍速播放
能标记已听
能批量生成 mock 音频
能重新生成音频
删除音频有二次确认
```

---

## 11. 最终通过标准

V4 最终通过：

```text
1. 所有 V4 必做验收通过。
2. V1/V2/V3 回归通过。
3. 前端无明显控制台错误。
4. 后端无启动错误。
5. 数据库旧数据可继续使用。
6. git status 中不包含 .env、data、node_modules、.venv、dist、__pycache__。
```
