# 09_环境变量与模型配置_V4

# AI Study Cast - V4 环境变量与模型配置

## 1. 配置原则

V4 不新增必须配置的真实密钥。

继续遵守：

```text
1. API Key 只能放 .env 或 app_settings。
2. 不写入代码。
3. 不写入 docs。
4. 不打印到日志。
5. 不返回前端。
```

---

## 2. AI 配置

继续支持：

```text
AI_BASE_URL
AI_API_KEY
AI_MODEL
AI_TIMEOUT_SECONDS
```

V4 推荐增加：

```text
AI_LOG_INPUT_MAX_LENGTH
AI_LOG_OUTPUT_MAX_LENGTH
```

默认值：

```text
AI_LOG_INPUT_MAX_LENGTH=1000
AI_LOG_OUTPUT_MAX_LENGTH=2000
```

---

## 3. TTS 配置

继续支持：

```text
TTS_PROVIDER=mock
XIAOMI_TTS_BASE_URL=
XIAOMI_TTS_API_KEY=
```

说明：

```text
真实小米 TTS 地址不明确时不要写死。
mock provider 必须可用。
```

---

## 4. 前端配置

前端只允许：

```text
VITE_API_BASE_URL=http://localhost:8000
```

禁止：

```text
VITE_AI_API_KEY
VITE_XIAOMI_TTS_API_KEY
```

---

## 5. 日志脱敏

AI 日志必须过滤：

```text
Authorization
Bearer token
api_key
secret
password
```

前端只展示：

```text
base_url
model_name
调用类型
状态
错误摘要
```
