# AI Study Cast - V4 阶段提示词合集

## 使用说明

每次只复制一个阶段给 AI。

每个阶段开始前必须说明：

```text
1. 当前阶段
2. 本阶段目标
3. 计划创建或修改的文件
4. 本阶段不会做哪些后续功能
```

每个阶段完成后必须说明：

```text
1. 本次完成了什么
2. 修改了哪些文件
3. 如何启动后端
4. 如何启动前端
5. 如何测试本阶段
6. 当前未完成或需要用户配置的内容
7. 下一阶段建议
```
