# V1 阶段概览

## 阶段定位

V1 是 AI Study Cast 的最小可运行闭环。目标是让本地系统完成从资料导入到 AI 知识点、题目、刷题、错题、TTS 音频播放的完整主流程。

## 已规划内容

V1 覆盖：

- 本地前后端项目结构。
- FastAPI 后端最小服务和健康检查。
- Vue 3 前端基础页面和路由。
- SQLite 数据库与核心表。
- 知识库、资料、知识点、题目、错题、音频、设置等基础业务对象。
- AI 知识点提取、AI 题目生成、AI 连接测试。
- TTS mock provider 和小米 TTS provider 预留。
- 音频静态文件访问。
- 基础启动、验收、错误提示、Loading、删除确认规则。

## 后续 AI 需要知道

- V1 优先级是“能运行、能保存、能调用、能展示”。
- 前端禁止直连 AI/TTS，所有外部能力必须走后端。
- 真实密钥只能放 `.env` 或数据库设置表，不能进代码和文档。
- 数据库文件必须在 `data/app.db`，不能放到 `backend/data/app.db`。
- 每次开发只推进一个阶段，不提前做后续功能。
