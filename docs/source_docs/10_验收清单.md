# 10_验收清单

# AI 知识点学习与音频播报系统 - V1 验收清单

## 1. 文档目的

本文档用于检查项目 V1 是否真正完成。

它主要解决：

```text
项目是不是能启动？
前端是不是能打开？
后端接口是不是正常？
数据库是不是能保存数据？
AI 提取知识点是不是可用？
题目生成是不是可用？
刷题和错题是不是可用？
TTS 音频是不是能生成和播放？
配置和密钥是否安全？
```

本清单适合在以下场景使用：

```text
每个开发阶段结束后检查
整体联调时检查
交给 Claude Code / Codex 修 bug 时检查
V1 最终验收时检查
```

---

## 2. 验收总原则

V1 成功标准不是代码看起来很多，而是：

```text
能启动
能访问
能保存
能调用 AI
能生成题目
能刷题
能生成音频
能播放音频
重启后数据不丢
```

不要只看文件是否生成，要实际运行检查。
