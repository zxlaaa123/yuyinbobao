# 02_技术架构_精简版

# AI 知识点学习与音频播报系统 - 技术架构设计

## 1. 文档目的

本文档定义项目 V1 的技术架构，重点说明：

```text
前后端如何分层
数据库和本地文件如何保存
AI 和 TTS 如何封装
接口如何组织
项目如何启动
```

架构原则：

```text
简单
稳定
本地可运行
方便扩展
```

---

## 2. 总体架构

项目采用：

```text
前后端分离 + 本地运行 + SQLite + 本地文件存储
```

调用关系：

```text
浏览器前端
↓ HTTP API
FastAPI 后端
↓
SQLite 数据库
↓
data 本地目录
↓
AI 接口 / TTS 接口
```

默认地址：

```text
前端：http://localhost:5173
后端：http://localhost:8000
```

---

## 3. 技术栈

### 前端

```text
Vue 3
Vite
TypeScript
Element Plus
Vue Router
Pinia
Axios
```

前端职责：

```text
页面展示
用户输入
表单校验
调用后端 API
显示 loading
展示错误提示
播放音频
```

前端不做：

```text
不直接访问数据库
不直接调用 AI API
不直接调用 TTS API
不保存真实 API Key
```

---

### 后端

```text
Python
FastAPI
Uvicorn
SQLAlchemy
SQLite
Pydantic
python-dotenv
httpx
```

后端职责：

```text
提供 HTTP API
连接 SQLite
保存业务数据

调用 TTS 接口
保存音频文件
暴露音频静态资源
读取配置
统一错误处理
```

---

### 数据库

```text
SQLite
data/app.db
```

V1 不使用：

```text
MySQL
PostgreSQL
Redis
```

### 本地文件

```text

├─ app.db

```text

配置项：

```text
AI_PROVIDER
AI_API_KEY
AI_BASE_URL
AI_MODEL
AI_TEMPERATURE
AI_TIMEOUT
```

```text

```

核心方法：

```text
test_ai_connection()

generate_questions()
```

### JSON 解析

必须支持：

```text
纯 JSON
Markdown ```json 代码块

```text
backend/app/utils/json_parser.py
```

## 11. TTS 架构

### 功能

```text
知识点 → 播报文本 → TTS → MP3 → data/audio → 前端播放
```

### Provider

V1 必须先支持：

```text
mock
```

再预留：

```text
xiaomi
```

不要在没有官方参数时写死小米 TTS 接口。

### 服务文件

```text
backend/app/services/tts_service.py
backend/app/services/audio_service.py
```

### 音频命名

```text
kp_{knowledge_point_id}_{timestamp}.mp3
```

### 访问路径

数据库保存：

```text
/audio/kp_1_20260506_153000.mp3
```

前端播放：

```text
VITE_API_BASE_URL + file_url
```

## 13. API 规则

业务接口统一前缀：

```text
/api
```

示例：

```text
GET /api/health
GET /api/knowledge-bases
POST /api/ai/extract-points
POST /api/tts/generate
```

音频静态文件不加 `/api`：

```text
/audio/xxx.mp3
```

错误返回 V1 可使用 FastAPI 默认格式：

```json
{
  "detail": "错误说明"
}
```

## 14. 关键流程

### 资料到知识点

```text
前端提交资料
↓
后端保存资料
↓
调用 AI
↓
解析 JSON
↓
保存知识点
↓
前端展示结果
```

### 知识点到题目

```text
选择知识点
↓
调用 AI 生成题目
↓
校验题目格式
↓
保存 questions
```

### 刷题到错题

```text
获取题目
↓
提交答案
↓
后端判断
↓
保存 answer_records
↓
答错写入 wrong_questions
```

### 知识点到音频

```text
读取知识点
↓
构建播报文本
↓
调用 TTS
↓
保存 MP3
↓
写入 audio_files
↓
前端播放
```

## 15. 启动方式

后端：

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

前端：

```bash
cd frontend
npm install
npm run dev
```

访问：

```text
http://localhost:5173
```

---

## 16. start.bat

根目录可提供：

```bat
@echo off
echo 启动 AI 知识点学习与音频播报系统...

start cmd /k "cd backend && .venv\Scripts\activate && uvicorn app.main:app --reload --host 127.0.0.1 --port 8000"
start cmd /k "cd frontend && npm run dev"

echo 后端地址：http://localhost:8000
echo 前端地址：http://localhost:5173
pause
```
