# 07_前端页面设计_精简版

# AI 知识点学习与音频播报系统 - 前端页面设计

## 1. 文档目的

本文档定义 V1 前端页面、路由、组件、交互和 UI 参考规则。

前端技术栈：

```text
Vue3 + Vite + TypeScript + Element Plus + Vue Router + Pinia + Axios
```

---

## 2. 前端设计目标

V1 优先：

```text
清晰
好用
流程完整
错误有提示
方便扩展
```

不要一开始追求复杂炫酷，先保证：

```text
能操作
能展示
能调用接口
长耗时有 loading
空数据有提示
```

---

## 3. UI 风格

推荐后台式布局：

```text
左侧菜单 + 右侧内容区
```

如果存在：

```text
docs/ui-prototype.html
```

它是 UI 视觉参考。

可以参考：

```text
蓝紫主色调
白色圆角卡片
左侧菜单
首页统计卡片
资料导入左右分栏
知识点卡片
刷题卡片
音频列表
设置双栏卡片
```

但正式前端必须写在：

```text
frontend/
```

不要直接在 HTML 原型上写业务逻辑。

# 9. 页面设计

## 9.1 首页 DashboardView

### 目标

展示系统概览和后端连接状态。

### 内容

```text
后端连接状态
知识库数量
资料数量
知识点数量
题目数量
错题数量
音频数量
快捷入口
```

### 接口

```http
GET /api/health
GET /api/dashboard/summary
```

### 错误

后端连接失败时显示：

```text
无法连接后端，请确认后端服务是否已启动。
```

---

## 9.2 知识库 KnowledgeBaseView

### 功能

```text
查看知识库列表
新建知识库
编辑知识库
删除知识库
```

### 显示字段

```text
名称
描述
资料数量
知识点数量
创建时间
操作
```

### 接口

```http
GET    /api/knowledge-bases
POST   /api/knowledge-bases
PUT    /api/knowledge-bases/{id}
DELETE /api/knowledge-bases/{id}
```

### 规则

```text
名称必填
删除前二次确认
非空知识库删除失败时显示中文提示
```

---

## 9.3 资料导入 MaterialImportView

### 功能

```text
选择知识库
输入标题
填写来源
粘贴正文
填写备注
保存资料
保存并提取知识点
```

### 表单字段

```text
所属知识库：必选
资料标题：必填
资料来源：选填
资料正文：必填
备注：选填
```

### 接口

```http
GET  /api/knowledge-bases
POST /api/materials
POST /api/materials/import-and-extract
```

### AI 提取交互

点击“保存并提取知识点”后：

```text
显示 loading
禁用按钮
成功后显示提取数量和知识点标题
失败后显示中文错误
```

## 9.6 刷题练习 PracticeView

### 功能

```text
选择知识库
选择知识点
选择题型
设置题目数量
开始练习
选择答案
提交答案
查看解析
下一题
显示结果
```

### 展示

```text
当前进度
题干
选项
提交按钮
正确答案
解析
下一题按钮
```

### 接口

```http
GET  /api/knowledge-bases
GET  /api/knowledge-points
GET  /api/practice/questions
POST /api/practice/answer
```

### 规则

```text
未选择答案不能提交
提交后显示对错和解析
提交后选项不可再次修改

```

# 10. Loading 规则

以下操作必须有 loading：

```text
页面加载数据
保存资料
保存并提取知识点
AI 生成题目
TTS 生成音频
提交答案
测试 AI 连接
测试 TTS 连接
删除数据
```

---

# 11. 错误提示

使用 Element Plus：

```text
ElMessage
ElNotification
ElMessageBox
```

常见错误：

```text
后端连接失败
AI API Key 未配置
AI 返回格式错误
TTS 生成失败
资料正文不能为空
请选择知识库
暂无题目
暂无音频
```

不要只显示：

```text
error
undefined
failed
```

# 12. 空状态

```text
暂无知识库，点击新建知识库开始。
暂无知识点，请先导入资料并提取。
暂无题目，请先生成题目。
暂无错题，继续保持。
暂无音频，请在知识点详情页生成音频。
```

# 13. 表单校验

知识库：

```text
name 必填，1 到 100 字
description 可选
```

资料：

```text
knowledge_base_id 必选
title 必填
content 必填
source 可选
note 可选
```

生成题目：

```text
count 范围 1 到 20
question_types 至少选择一种
```

设置：

```text
AI Temperature 范围 0 到 2
API Key 可以为空，但测试连接会失败
```
