# AI Study Cast - V3 前端页面设计

## 1. V3 前端目标

```text
结构清晰
交互一致
错误可理解
学习流程更顺
```

不做大改版。

---

```text
/wrong-questions

/settings
```

---

## 3. 建议新增页面

```text
/backups
/search
/ai-logs
/study-sessions
```

菜单建议：

```text
备份恢复
```

```text

```

## 4. 布局优化

建议新增：

```text
layouts/MainLayout.vue
```

App.vue 只保留：

```text
router-view
```

MainLayout 负责：

```text
侧边栏
顶部状态栏
主题切换
后端状态
```

## 5. 路由优化

新增：

```text
frontend/src/router/index.ts
```

main.ts 只做：

```text
createApp
use router
use ElementPlus
mount
```

---

## 6. 类型优化

新增：

```text
frontend/src/types/
```

迁移原则：

```text
1. 多个页面共享的类型放 types/
2. 只在一个 API 文件使用的类型可暂时保留
3. 不一次性搬所有类型
```

---

## 7. 错误提示

前端统一方法：

```text
getErrorMessage(error, fallback)
```

展示优先级：

```text
error.response.data.detail
error.response.data.message
fallback
```

必须是中文。

---

## 8. Loading

以下操作必须有 loading：

```text
创建备份
恢复备份
AI 提取
AI 生成题目
AI 生成闪卡
TTS 生成
删除数据
导入文件
搜索
```

---

## 9. 体验原则

```text
操作入口清楚
空状态明确
危险操作二次确认
表格支持筛选
移动端能基本使用
```
