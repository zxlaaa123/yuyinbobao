# 05_AI与提示词设计_V3

# AI Study Cast - V3 AI 与提示词设计

## 1. V3 AI 目标

V3 不新增复杂 AI 架构。

重点是：

```text
更稳定
可排查
可限制
可复用
```

---

## 2. AI 调用日志

每次 AI 调用记录：

```text
call_type
provider
model
status
input_chars
output_chars
duration_ms
error_message
created_at
```

call_type：

```text
extract_points
generate_questions
generate_flashcards
repair_json
```

禁止记录：

```text
API Key
Authorization header
完整敏感资料
```

---

## 3. Prompt Profile

V3 可支持提示词配置档。

初期固定内置：

```text
default
exam-focused
memory-focused
brief
```

用途：

```text
default：均衡
exam-focused：偏考点和题目
memory-focused：偏记忆方法
brief：短输出，降低 token 消耗
```

---

## 4. JSON 稳定性

继续支持：

```text
纯 JSON
Markdown JSON 代码块
前后有额外文本的 JSON
简单格式修复
AI 二次修复
```

V3 建议补充：

```text
1. JSON schema 级字段校验
2. 缺字段默认值
3. 多余字段忽略
4. 每次解析失败写入 ai_call_logs
```

---

## 5. AI 调用上限

建议配置：

```text
AI_MAX_INPUT_CHARS
AI_MAX_OUTPUT_ITEMS
AI_ENABLE_CALL_LOG
AI_LOG_BODY_PREVIEW_CHARS
```

默认：

```text
AI_MAX_INPUT_CHARS=20000
AI_MAX_OUTPUT_ITEMS=30
AI_ENABLE_CALL_LOG=true
AI_LOG_BODY_PREVIEW_CHARS=500
```

---

## 6. 不做内容

V3 暂不做：

```text
复杂 Agent
多轮自动规划
向量 RAG 问答
联网搜索学习资料
自动上传云端
```
