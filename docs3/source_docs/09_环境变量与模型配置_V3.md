# 09_环境变量与模型配置_V3

# AI Study Cast - V3 环境变量与模型配置

## 1. 原则

```text
真实 API Key 只能放 .env 或 app_settings
不能写进代码
不能写进 docs
不能暴露给前端
```

---

## 2. 保留配置

继续保留：

```text
AI_PROVIDER
AI_API_KEY
AI_BASE_URL
AI_MODEL
AI_TEMPERATURE
AI_TIMEOUT
AI_SEGMENT_SIZE
AI_MAX_SEGMENTS

TTS_PROVIDER
XIAOMI_TTS_API_KEY
XIAOMI_TTS_BASE_URL
XIAOMI_TTS_VOICE
XIAOMI_TTS_FORMAT
XIAOMI_TTS_SPEED

VITE_API_BASE_URL
```

---

## 3. V3 建议新增配置

```text
BACKUP_DIR=data/backups
BACKUP_MAX_COUNT=20

AI_ENABLE_CALL_LOG=true
AI_LOG_BODY_PREVIEW_CHARS=500
AI_MAX_INPUT_CHARS=20000
AI_MAX_OUTPUT_ITEMS=30

SEARCH_ENABLE_FTS=false
SEARCH_MAX_RESULTS=50

REVIEW_OVERDUE_DAYS=1
REVIEW_DAILY_LIMIT=30

TTS_DEFAULT_FORMAT=mp3
TTS_REGENERATE_KEEP_OLD=true
```

---

## 4. .env.example 建议

V3 更新 `.env.example` 时加入：

```text
BACKUP_DIR=data/backups
BACKUP_MAX_COUNT=20
AI_ENABLE_CALL_LOG=true
AI_LOG_BODY_PREVIEW_CHARS=500
AI_MAX_INPUT_CHARS=20000
AI_MAX_OUTPUT_ITEMS=30
SEARCH_ENABLE_FTS=false
SEARCH_MAX_RESULTS=50
REVIEW_OVERDUE_DAYS=1
REVIEW_DAILY_LIMIT=30
TTS_DEFAULT_FORMAT=mp3
TTS_REGENERATE_KEEP_OLD=true
```

---

## 5. 前端配置

前端只允许：

```text
VITE_API_BASE_URL=http://localhost:8000
```

禁止：

```text
VITE_AI_API_KEY
VITE_TTS_API_KEY
VITE_XIAOMI_TTS_API_KEY
```
