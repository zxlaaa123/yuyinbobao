# 11_阶段提示词合集

# AI Study Cast - 每个开发阶段可复制提示词合集

## 1. 文档用途

本文档专门用于在 VSCode、Claude Code、Codex、Copilot Chat、Cursor、Trae 等 AI 编程工具中复制使用。

你可以按阶段把提示词发给 AI，让它一步一步开发项目。

本项目名称：

```text
AI Study Cast
AI 知识点学习与音频播报系统
```

项目形式：

```text
本地运行 Web 应用
启动后端 + 启动前端 + 浏览器访问
```

前端：

```text
Vue3 + Vite + TypeScript + Element Plus
```

后端：

```text
Python + FastAPI + SQLite + SQLAlchemy
```

---

请先把文档放到项目目录中，推荐结构：

```text

│  ├─ 07_前端页面设计.md
│  ├─ 08_开发阶段计划.md
│  ├─ 09_环境变量与模型配置.md
```


是静态前端视觉原型，只作为 UI 风格、布局、页面结构参考。

正式前端必须写在：


不能直接在 `ui-prototype.html` 上堆业务逻辑。

# 27. 最推荐的使用方式

每次只复制一个阶段的提示词。

不要一次性把 18 个阶段都发给 AI。

推荐流程：

```text
复制阶段 0 提示词
↓
AI 完成
↓
你运行测试
↓
通过后复制阶段 1 提示词
↓
继续
```

如果你只是说“继续”，AI 可能会一次做太多，容易跑偏。
